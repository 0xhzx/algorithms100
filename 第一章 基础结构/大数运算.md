## 实现指数函数

```python
def pow(a,b):
    if b==0:
        return 1
    if b==1:
        return a
    t=a//2
    return pow(a,t)*pow(a,t)*pow(a,b-2*t)

def power(a,b):
    re=1
    base=a
    while b:
        if b&1:
            re*=base
        base*=base
        b>>1
    return re
```
